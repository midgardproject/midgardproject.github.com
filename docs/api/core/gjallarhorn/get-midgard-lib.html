Getting Midgard

  • The easiest way to download Midgard is to use already built binary packages. For Vinland (Midgard2 9.03) you may find some at OBS repository.

  • You can download release tarballs from Midgard website.

  • Also, you can get Midgard from SVN repository. 

          $ mkdir -p svn/midgard/branches
      $ cd svn/midgard/branches
      $ svn co https://svn.midgard-project.org/midgard/branches/vinland
